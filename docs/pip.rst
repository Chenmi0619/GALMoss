.. _pip:

How to install 
===============
We provide two kinds of methods to download and install GalMOSS, ``pip`` and ``git``.

Install via pip
---------------

.. code-block:: bash

    pip install galmoss


Install via git
---------------

.. code-block:: bash

    git clone https://github.com/Chenmi0619/GALMoss
    cd galmoss
    python setup.py install
